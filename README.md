# Task Schedule

Anwendung um Kommando Ereignisse zu spezifischen Zeiten auszulösen.

## Installation

#### Vorraussetzungen:

- node 10 oder neuer
- npm

#### Installieren

```bash
git clone https://github.com/playduck/Task-Schedule.git
npm install
```

#### Bauen

```bash
npm run build
```

optional noch

```bash
npm run minify
```

#### Ausführen

```bash
npm run start
```
